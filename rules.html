General Rules

1. Dilarang menggunakan Cheat, Mod Ilegal, atau Program Pihak Ketiga.

2. Wajib menggunakan nama sesuai dengan culture. Cth: Alexander_Bradwey

3. Dilarang Spam, Flood, dan Promosi Server lain.

4. Menybarkan identitas tanpa izin akan dikenakan punishment.

5. Saling menghormati sesama Player dan Admin.

Break Rules

1. POWERGAMING (PG)

Melakukan tindakan yang tidak mungkin bisa dilakukan di dunia nyata.

-> Contoh: Mobil seat 4 dipakai untuk 6 orang.


2. METAGAMING (MG)

Menggunakan informasi OOC untuk kepentingan pribadi.

-> Contoh: Melihat ID dan melihat nama dari ID pada tab list player, padahal tidak kenal atau belum pernah berkenalan sebelumnya.


3. DEATHMATCHING (DM)

Melakukan penyerangan / pembunuhan pemain lain tanpa RP yang jelas.

-> Contoh: Menembak orang tanpa RP jelas.


4. REVENGE KILL (RK)

Melakukan pembalasan / mengungkit kembali kejadian sebelum pingsan.

-> Contoh: Kamu ditembaki kelompok A, kamu kalah. Saat terbangun kamu "Ayo kita serang kelompok A lagi".
